## Learning objectives

- Understand core concepts of big data and their applications in healthcare
- Set up and use Python and key libraries for data ingestion and analysis (e.g., Pandas, NumPy, Matplotlib)
- Acquire, clean, and preprocess real-world health datasets for downstream analysis
- Build simple GUI tools with Tkinter and integrate data storage using SQLite
- Complete a project demonstrating end-to-end data handling and presentation

## Assessment & grading

Assessment will include assignments, project work, quizzes, and participation. For detailed criteria and rubrics, refer to the Course-Outline PPT linked above.
